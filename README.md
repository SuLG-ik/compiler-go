## О проекте

Это графическая оболочка языкового процессора, написанная на Go с использованием фреймворка Fyne. Приложение представляет собой текстовый редактор с возможностью запуска синтаксического анализатора: слева располагается область ввода кода, справа — панель вывода результатов. Интерфейс поддерживает стандартные операции с файлами, редактирование текста и навигацию через меню, панель инструментов и горячие клавиши.

## Сборка и запуск

### Зависимости

Нужен установленный Go (проверялось на 1.26).

### Установка зависимостей Go

```bash
go mod tidy
```

### Сборка

```bash
go build -o compiler .
```

После сборки исполняемый файл `./compiler` появится в корне проекта.

### Путь к исполняемому файлу

Готовые бинарники публикуются на странице [Releases](https://github.com/SuLG-ik/compiler-go/releases/latest) в GitHub. Там же можно найти файлы для конкретных тегов:

```
https://github.com/SuLG-ik/compiler-go/releases/tag/<версия>
```

### Запуск

```bash
./compiler
```

## Интерфейс и функции

Окно приложения состоит из четырёх зон: строка меню, панель инструментов, рабочая область (редактор + вывод) и строка состояния.

![Главное окно](docs/main_window.png)

### Панель инструментов

Расположена вверху под строкой меню. Содержит кнопки-иконки для самых частых действий: создать, открыть, сохранить, редактировать, запустить анализатор и справку.

![Панель инструментов](docs/toolbar.png)

### Меню «Файл»

![Меню Файл](docs/menu_file.png)

- **Создать** (`Ctrl+N`) — очищает редактор, сбрасывает текущий файл
- **Открыть** (`Ctrl+O`) — открывает диалог выбора `.txt`-файла
- **Сохранить** (`Ctrl+S`) — сохраняет текущий файл; если файл новый, работает как «Сохранить как»
- **Сохранить как** (`Ctrl+Shift+S`) — сохраняет в новый файл по выбору
- **Выход** — закрывает приложение

### Меню «Правка»

![Меню Правка](docs/menu_edit.png)

Стандартные операции редактирования. Все они доступны и через контекстное меню редактора.

- **Отменить / Повторить** (`Ctrl+Z` / `Ctrl+Y`)
- **Вырезать / Копировать / Вставить / Удалить** (`Ctrl+X` / `Ctrl+C` / `Ctrl+V`)
- **Выделить всё** (`Ctrl+A`)

### Меню «Пуск»

- **Пуск** (`Ctrl+R`) — запускает синтаксический анализатор. Введённый в редакторе текст передаётся на обработку, результат появляется в правой панели вывода.

### Рабочая область

![Рабочая область](docs/editor.png)

Делится на две части вертикальным разделителем, который можно перетаскивать мышью:

- **Редактор (слева)** — многострочное текстовое поле для ввода исходного кода. Поддерживает все стандартные операции редактирования.
- **Вывод (справа)** — область «только чтение», в которую языковой процессор записывает результат анализа.

### Строка состояния

Расположена внизу окна. Отображает текущее состояние приложения (например, путь к открытому файлу или сообщение о результате выполнения).

## Ограничения

- На данном этапе реализована только графическая оболочка; логика синтаксического анализатора подключается в следующих лабораторных работах. Все обработчики действий сейчас являются заглушками.
- Фильтр файлов при открытии/сохранении ограничен форматом `.txt`.
- Приложение не поддерживает вкладки — одновременно открыт только один файл.
- Fyne не поддерживает подсветку синтаксиса из коробки; в текущей реализации редактор является обычным текстовым полем.
